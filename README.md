## About This Repository

This repo serves as a public learning journal. Each folder corresponds to a course day and contains:

- **`task.py`** тАФ the exercise prompt with starter code
- **`solution.py`** тАФ my completed solution
- **`main.py`** тАФ project entry point (for larger projects)
- Supporting files (`art.py`, `game_data.py`, etc.)

If you are also taking this course, feel free to use this as a reference тАФ but I encourage you to attempt each exercise on your own first.

## How to Run the Code

1. **Clone the repository:**
   ```bash
   git clone https://github.com/danu52-blip/100-days-of-python.git
   cd 100-days-of-python
   ```

2. **Navigate to any day:**
   ```bash
   cd "Day 5/Password Generator Project"
   ```

3. **Run a script:**
   ```bash
   python solution.py
   ```

> No external dependencies required for Days 1тАУ15. Pure Python standard library.
